Download

Para fazer o download do instalador clique no link abaixo:

Download AtualizaTabwin

Depois de instalado o aplicativo é atualizado automaticamente em cada execução

Requisitos mínimos:

Sistemas operacionais

  • Windows XP
  • Windows Vista
  • Windows 7
  • Windows 8
  • Windows Server 2003
  • Windows Server 2008
  • Windows Server 2012

Há apenas a versão 32 bits do AtualizaTabwin, que pode ser usada no Windows 32 e 64 bits.

Java

  • Requer o Java 1.7 32bits. Para obter clique aqui: www.java.com

Hardware mínimo

  • Pentium 4 ou mais recente
  • 1GB de RAM
  • 200MB de espaço em disco

Acesso a Internet

Você precisa de acesso liberado para os seguintes endereços:

  • http://www.atualizatabwin.com.br:80
  • http://update.atualizatabwin.com.br:80
  • ftp://ftp.datasus.gov.br:21 (FTP Passivo é necessário)
